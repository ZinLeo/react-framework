# React-framework

  React project skeleton based on Create-react-app, React 16.5.2

## Feature

1、Support for preprocessor scss

## Use

1、Clone project
  
    git clone https://github.com/ZinLeo/react-framework.git

2、Installation dependence

    1、npm install
    
    2、yarn

3、Start

    npm start

4、Unpack

    npm run build
